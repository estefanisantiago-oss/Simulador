# Simulador 3D de Laboratorio – Three.js

## Objetivo del Proyecto
Desarrollar una aplicación web interactiva en 3D utilizando Three.js que permita visualizar objetos geométricos con distintos tipos de iluminación, relleno, texturas y técnicas de sombreado, permitiendo al usuario interactuar con los modelos en tiempo real.

---

## Descripción del Proyecto
El sistema permite:
- Insertar figuras 3D (cubo, esfera y cilindro).
- Activar y desactivar distintos tipos de luces.
- Aplicar color sólido y degradado.
- Aplicar texturas.
- Cambiar técnicas de sombreado.
- Activar modo wireframe.
- Activar y desactivar rotación automática.
- Manipular rotación, traslación y escalado mediante sliders.

---

## Instrucciones de Uso
1. Abrir el archivo `index.html` en un navegador web.
2. Usar el panel lateral para agregar una figura.
3. Activar las luces deseadas.
4. Aplicar colores, degradados o texturas.
5. Probar los distintos tipos de sombreado.
6. Modificar rotación, posición y escala de la figura.
7. Usar el botón de rotación para animar el objeto.
